# Folder Hierachy
+```
~
	~/.vim/
		~/.vim/templates : for Template skeleton files
		~/.vim/plugged   : Git repos for plugin
		~/.vim/autoload	 : Folder for files to autoload
```

# Vim Controls
## Movement
```
h : Move up
j : Move left
k : Move right
l : Move down
```

## Tabs
```
:tabprev
:tabnext
:tabfirst
:tablast
```
