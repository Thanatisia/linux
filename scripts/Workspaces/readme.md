# Background
```
This folder is for scripts that generate the template for the appropriate languages
```

# Examples
```
Script name: generate_html5.sh
Language: html5
Output:
<html>
	<head>
		<title>Hello World</title>
	</head>
	<body>
		<p>
			<h1>Hello World</h1>
		</p>
	</body>
</html>
```
